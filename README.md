# NodeJS RESTful API Boilerplate 

### Clone repository
```git clone https://github.com/Pelegrinetti/my-nodejs-boilerplate.git nodejs-api```
### Go to folder
```cd nodejs-api```
### Install dependencies
```yarn install```
### Copy *.env.example* to *.env*
```cp .env.example .env```
### Set your environment variables
### Run your API:
```yarn start```
#### Or in development:
```yarn dev```
